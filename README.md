# Task Manager
## Task Manager basit bir görev yöneticisi uygulamasıdır.
Uygulamada hesap açtıktan sonra tarih seçip yapmanız gerekenleri kaydedebilir, okuyabilir, silebilirsiniz.

## Kurulum
```
npm install
```
## Çalıştırma
```
npm start
```
